## 0. 概述
+ 本项目主要是对`TSM`项目的二次开发。
  + 原始项目的介绍可以查看 [这里](docs/original_README.md)
+ 本项目主要包括以下内容：
  + [数据相关介绍](docs/DATA.md)：包括数据集预处理、模型训练所需数据集构建、模型所需数据预处理等。
  + [模型相关介绍](docs/MODEL.md)：包括模型结构、shift操作等介绍，还包括在线训练模型的基本介绍。
  + [模型移植介绍](online_demo/README.md)：如何将在线模型转换为TVM形式。
